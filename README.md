## Project Description
This project explores open-source software concepts through both theory and practical implementation. The Linux Kernel was selected due to its importance in modern computing.

The project includes analysis of its origin, licensing, ethical aspects, Linux environment footprint, and its role in the FOSS ecosystem. Shell scripting tasks were also performed to understand automation in Linux systems.

## Scripts Overview

### Script 1: System Identity Report
Displays system details like kernel version, user, uptime, and environment.

### Script 2: FOSS Package Inspector
Checks the status of a software package and displays basic information.

### Script 3: Disk and Permission Auditor
Analyzes important directories and shows permissions.

### Script 4: Log File Analyzer
Analyzes a sample log file and counts occurrences of keywords like "error".

### Script 5: Open Source Manifesto Generator
Generates a personalized open-source statement using user input.

## How to Run Scripts

1. Open a bash environment (Linux / Online Bash)
2. Make script executable:
   chmod +x script_name.sh
3. Run script:
   ./script_name.sh
